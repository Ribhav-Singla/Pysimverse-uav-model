🚨 IF SOMETHING BREAKS
═════════════════════════════════════════════════════════════════════════════

Error: "ModuleNotFoundError: No module named 'ultralytics'"
  → pip install ultralytics

Error: "CUDA out of memory"
  → Set 'yolo_device': 'cpu' in CONFIG

Error: Import fails "No module named 'yolo_depth_detector'"
  → Make sure yolo_depth_detector.py is in same directory as uav_env.py

Error: "Camera 'uav_depth_camera' does not exist"
  → Check that environment.xml has <camera name="uav_depth_camera"/>

Error: "All features are zero"
  → Try lowering yolo_confidence_threshold to 0.3

💡 TIPS & TRICKS

🎓 UNDERSTANDING YOLO FEATURES (64D Vector)
═════════════════════════════════════════════════════════════════════════════

The 64D vector contains:
  [0:20]   Obstacle detection features (distance, angles, confidence)
  [20:36]  Goal detection features
  [36:48]  Boundary detection features
  [48:64]  Detection statistics & confidence metrics

All values normalized to [0, 1] for stable training.
