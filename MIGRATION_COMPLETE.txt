🎓 UNDERSTANDING THE NEW SYSTEM
═════════════════════════════════════════════════════════════════════════════

OLD FLOW (CNN):
  Depth Image → Preprocess → CNN (3 conv + 2 MLP) → 128D features
                             (requires training)
                             (learned patterns, no meaning)

NEW FLOW (YOLOv8):
  Depth Image → Normalize → RGB Conversion → YOLO Detection
                           ↓
                      Parse Boxes → Extract Spatial Relations
                           ↓
                      64D Spatial Features
                      (distance, angles, confidence, detection flags)

YOLO 64D Feature Vector Structure:
  [0:20]   - Obstacle Detection (4 closest obstacles)
  [20:36]  - Goal Detection (3 closest goals)
  [36:48]  - Boundary Detection (4 edge distances + metrics)
  [48:64]  - Detection Statistics (confidence scores, counts)

All values normalized to [0, 1] for training stability.

   Check YOLO detections: Print env.depth_extractor.detector.last_detections
   Profile inference: env.depth_extractor.detector.profile_inference(image)

4. Convergence Issues:
   Try increasing yolo_confidence_threshold to 0.3 (more detections)
   Or decreasing to 0.7 (fewer false positives)

5. GPU Memory Issues:
   Use yolo_device: 'cpu'
   Or switch to nano model (yolov8n)
